             2006/07/01

1. 概要
---------

  Qt-LCoV は、回路図エディタ Qt-BSch3V のための部品ライブラリ作成ソフトウエア
です。
  コンパイルには Qt 4.1.2 が必要です。
  Qtについては、Trolltech社のWEBサイト(http://www.trolltech.com)をご覧くださ
い。


2. ファイル構成
-----------------

  COPYING          ライセンス文書
  readme-j.txt     このファイル
  qtlcov/          ソースファイル
  
  ***添付しているソースの文字コードはEUC-JPです。***


3. 開発環境とコンパイル
-------------------------

  圧縮ファイルには実行形式を含みませんのでコンパイルしていただく必要があり
ます。

  次の環境で開発と動作確認を行いました。

 Windows XP pro SP2 + Qt 4.1.2 opensource edition
 Knoppix Linux 3.8.1 on HDD + Qt 4.1.2 opensource edition
 
コンパイルはコマンドラインから次の手順で行います。

$cd qtlcov
$qmake
$make
これで、実行ファイル qtlcov が得られます。

qtlcov.proファイルは、提供しているものをお使いください。

4. 使い方
-----------

部品を作成する作業の流れを示します。
(1)Editメニューから New Component を選んで、部品のプロパティを設定
  このとき「Use pattern」をチェックすると、模様を持つ部品を作れます。
(2)右側のウィンドウが部品の編集画面です。
  拡大縮小は、Viewメニューで行えます。(PageUp, PageDownキーでもできます)
(3)ツールを使って部品を編集します。
 ・Pin
    クリックすると配置します。
    配置したときは赤色で表示。このときに、編集のプロパティ(または、ピンを囲
    む点線内でダブルクリック)で、ピン名や番号を設定できます。
    ピンを囲む点線内でマウスの左ボタンを押下して移動すると、ピン移動ができ
    ます。すでにピンのあるところには移動できません。
 ・Line
    クリックして始点、もう一度クリックして終点。
    ツールメニューのノード編集モードでは、線の単独選択状態で端点の移動ができ
    ます。
    単独選択状態において、編集のプロパティで線幅と実線破線を設定できます。
 ・Ellipse
    円、楕円。
    クリックして始点、もう一度クリックして終点。始点、終点で囲まれた四角形に
    内接する円、楕円を配置します。
    ツールメニューのノード編集モードでは、円の単独選択状態で変形ができます。
    単独選択状態において、編集のプロパティで線幅と実線破線、塗りつぶし状態を
    設定できます。
 ・Polygon
    クリックするたびに頂点を配置します。
    最後の頂点でダブルクリックしてください。
    いったん配置した多角形の頂点の数の変更はできません。
    ToolメニューのMove Nodeでは、多角形の単独選択状態で変形ができます。
    単独選択状態において、編集のプロパティで線幅と実線破線、塗りつぶし状態を
    設定できます。
 ・Arc
    クリックで中心。マウスを水平に動かしてクリックして直径の決定。
    3回目のクリックで円弧始点、4回目のクリックで円弧終点。
    ツールメニューのノード編集モードでは、円弧の単独選択状態で変形ができます。
    単独選択状態において、編集のプロパティで線幅と実線破線を設定できます。
 ・Text
    クリックで配置位置を決定。
    ダイアログでテキストの設定。
 ・Select
    クリックで単独選択、ドラッグで複数選択。
    ピンは単独選択しかできません。
    複数選択は選択対象がすべて囲まれている状態のときに選択されます。

  Editメニューには、「Copy」と「Copy Component」があります。
  前者は、ピンやパターンのコピー、後者はQt-LCoVを複数立ち上げて、Qt-LCoV間で
部品をやりとりするのに使います。Qt-BSch3Vと直接部品をコピーペーストすること
もできます。

  ToolメニューのCreate Bit Pattern の作成を選ぶと、配置したベクトルデータか
ら、以前の回路図エディタ Qt-BSch で扱うビットパターンを作成します。Qt-BSch3V
でないQt-BSchや、BSch3、BSchでライブラリを使う場合はビットパターンを作成して
ください。
